# LangChain + Ollama Chatbot

## Description
This is a simple AI chatbot built using Streamlit, LangChain, and Ollama (Llama 3 model).

## Features
- Ask questions to AI
- Real-time responses
- Simple UI using Streamlit

## Tech Stack
- Python
- Streamlit
- LangChain
- Ollama (Llama 3)

## How to Run

1. Install Ollama from https://ollama.com
2. Run model
